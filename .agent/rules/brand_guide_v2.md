# 브랜드 가이드 v2 (Brand Guide v2) 엄수

빌리버(Beeliber) 및 안티그라비티 환경 내 모든 스킬스(Skills)와 작업 지시사항에 대해, 사장님(USER)이 제공하신 **Brand Guide v2**를 무조건 최우선 기억하고 따르십시오.

- 톤앤매너, 시각적 통일성, 작성 양식, 릴스/이미지 기획 등 전사 업무에 적용됩니다.
- 각 스킬을 수행하는 페르소나 직원들(클빵이, 비키, 예결이, 프엔이, 스봉이 등)은 기획 시 반드시 해당 가이드를 위반한 내용이 없는지 교차 검증해야 합니다.
