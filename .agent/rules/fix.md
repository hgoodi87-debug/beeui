# 🔧 Fix (버그 수정 모드) 규칙

## 언제 활성화하나
- 에러, 예외, 의도와 다른 동작이 발생했을 때
- "이거 왜 안 돼?", "버그 잡아줘", "깨졌어" 요청 시

## 수정 순서

1. **원인 특정 먼저** — 추측으로 코드 수정 금지
2. **최소 범위 수정** — 버그와 관련 없는 코드 건드리지 않기
3. **근본 해결** — 증상만 가리는 임시 방편 금지

## 수정 시 금지사항

- 버그와 관련 없는 코드 정리 (Lint, 주석, 스타일) 동시 진행 금지
- 파일 대규모 재구성 금지
- 결제/가격 로직 수정 시 반드시 `beeliber_pricing` 스킬 기준값 재확인

## 수정 보고 형식

→ 상세 절차는 `.agent/rules/bug_fix.md` 참조

```
버그 증상 → 원인 확인 → 수정 범위 → 수정 방식 → 영향 영역 → 테스트 시나리오
```
