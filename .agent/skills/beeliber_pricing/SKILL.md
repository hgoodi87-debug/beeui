## 🧮 계산 로직 공식

```
t = 이용 시간 (시간 단위, 분은 올림 처리)

if 0 < t <= 4:
    price = base_4h

elif 4 < t < 24:
    price = base_4h + ceil(t - 4) * hourly_after_4h

elif t == 24:
    price = price_day1

elif t > 24:
    price = price_day1 + ceil((t - 24) / 24) * extra_day_storage
```

**시간 단가 산정**:
- 손가방: (8,000 - 4,000) / 20 = **200원/시간**
- 캐리어: (10,000 - 5,000) / 20 = **250원/시간**
- 유모차·자전거: (14,000 - 10,000) / 20 = **200원/시간**

> [!IMPORTANT]
> **보험료는 첫날에만 1회** — 24시간 초과 추가 일자에는 보관비만 누적

## ⚠️ 운영 예외 처리

- 규격 초과인데 캐리어로 잘못 접수된 경우 → 운영자 수정 가능
- 위험물·냉장물 → 접수 차단
- 지점 공간 초과 시 → 예약 불가
- 손가방 단독 배송 → 불가 (캐리어 동반 필수)
