## 1. 브랜드 정체성

### 1.1 브랜드 네임 의미
- **Bee + Liber**: 부지런한 꿀벌(Bee)이 여행객에게 자유(Liber)를 배달한다는 의미
- **Believe**: 고객이 짐을 믿고 맡길 수 있는 '신뢰'의 가치
- **Delivery**: '빌리버'라는 발음을 통해 서비스 본질인 '배송'을 직관적으로 연상

### 1.2 Mission
**대한민국 여행의 무게를 덜고, 여행의 본질을 되찾기**

- **물리적 해결**: 무거운 짐으로부터 자유를 제공하여 여행의 질 높이기
- **경험적 가치**: 여행의 순간과 동선을 가볍게 만들어, 대한민국 여행의 진짜 가치에 집중하게 하기
- **동반자 역할**: 한정된 시간 안에서 여행자가 더 깊고 편안하게 머물 수 있도록 돕는 동반자의 역할

### 1.3 Vision
**한국 여행의 첫 관문이자 글로벌 여행 물류의 표준**

- **3-Year Goal**: 3년 이내, 한국을 방문하는 전 세계 여행가의 80%가 사용하는 필수 인프라 구축
- **Expansion**: 단순 배송 서비스를 넘어, 여행자의 이동 경로를 최적화하는 글로벌 여행 물류 플랫폼으로 진화

### 1.4 Core Value

| 키워드 | 가치 정의 |
|--------|----------|
| **Trust (절대적 신뢰)** | 고객의 짐은 단순한 물건이 아니라 '여행의 오늘'입니다. 안전과 정시성을 훼손할 수 없는 우리의 기본가입니다. |
| **Freedom (자유로운 경험)** | 우리는 짐을 옮기는 것이 아니라, 고객에게 '무게가 사라진 시간과 자유'를 배송합니다. |
| **Satisfaction (만족)** | 짐짝 서비스의 심리적 편안함과 해방감입니다. 고객의 만족을 충족시키는 서비스. |

### 1.5 브랜드 키워드

| 핵심 키워드 | 감성 키워드 | 기능 키워드 | 금지 키워드 |
|------------|-----------|-----------|-----------|
| 짐 없는 자유 | 설렘 / 해방감 | 실시간 추적 | 무거운 / 불편한 |
| 보이는 신뢰 | 여사 / 편안함 | 보험 보장 | 위험한 / 불안한 |
| 비일상 여행 | 가벼움 / 자유로움 | 문전배송 | 번거로운 / 복잡한 |
| 온전한 몰입 | 재미 / 행복 | 다국어 지원 | 스트레스 / 피곤한 |

### 1.6 경쟁사 대비 포지셔닝

#### vs Bounce / Radical (글로벌 보관 서비스)
- 거점 수 대등 마지만 **아시아권 여행자 중심**, 대만권 여행자에게 인터페이스·커뮤니케이션 불친화
- **24시간 단위 대여만 가능** → 빌리버는 4시간 단위로 더 유연
- **배송 서비스 없음** → 빌리버는 Hub→인천공항 비일 배송 제공
- 빌리버 핵심 타겟: 중화권·대만권·일본어권 (Bounce/Radical이 공략하지 않는 시장)

#### vs 짐캐리
- **가격이 더 비쌈** (짐캐리 7,000~17,000원 vs 빌리버 6,000~10,000원)
- **빌리버 거점이 더 많음** (파트너 33개+ vs 짐캐리)

#### 핵심 포지셔닝 메시지
보관 + 배송을 동시에 제공하는 서비스. 중화권·대만권 여행자를 위한 친화적 경험. 4시간 단위 유연한 요금.

> ⚠️ AI 주의: 거점 수 1위 표현 절대 금지 (Bounce 거점이 더 많음). 가격 비교는 범위로만.

### 샤오홍슈 이용 가이드 — AI 콘텐츠 작성 시 필수 참조

**현황 및 문제**
- 이벤트 참여자는 많으나 웹사이트 전환 없음
- 전정 고객 전체가 한 지점으로만 방문 (온라인 예약 없음)
- 이유: 게시물에 서비스 설명·예약 URL이 없어 서비스 자체를 모름

**샤오홍슈 AI 콘텐츠 2가지 필수 방향**

① **이벤트 + 서비스 설명 결합형**
이벤트 참여 의도 높이면서 "빌리버가 뭔지"를 게시물 안에 설명.
룰렛 이벤트 → 비짓 혜택 → "이 서비스가 이런 거야" → bee-liber.com 예약 전 안내로 구성.

② **URL 직접 노출 필수**
샤오홍슈는 링크 클릭이 제한적이므로 URL을 텍스트로 직접 게시물 본문에 표기.
- 본문 마지막: `🐝 bee-liber.com` 반드시 포함
- 프로필 bio에도 URL 명시 권장

**샤오홍슈 콘텐츠 특성**
- 한 지점 중심으로 작성 (현재 연남 지점 집중 지역)
- 중국어 간체 사용 (본토 중국 타겟)
- 이벤트 참여 → 서비스 인지 → 웹사이트 예약 플로우로 연결
- 이미지: 한 지점 실제 공간감 강조, 밝고 친근한 키

### 현재 마케팅 현황
- Instagram: 213명 (@beeliber_official)
- META Ads: 실행 준비 중 (Set A: 繁體中文 TW/HK / Set B: 영어권+SEA) → Traffic 목표, 50+ Purchase 이벤트 후 전환 최적화
